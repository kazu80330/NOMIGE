# Claude Code 実装指示プロンプト（Step 2用）

Viteプロジェクトのリファクタリングが完了したので、次に「ドボン（大富豪・ヤニブ融合版）」の実装を行ってください。以下の手順で進めてください。

## 1. 指示書の読み込み
新しく作成された `dobon_instructions.md` を読み込み、特に「大富豪ベースの強さ判定」と「手札の数字の種類数に応じたドボン計算ロジック」を理解してください。

## 2. コアロジックの実装（Step 2）
`src/games/dobon/` ディレクトリを作成し、以下の機能を実装してください。
- **DaifugoEngine**: カードの強さを比較し、スート縛りや枚数一致のバリデーションを行う。
- **AdvancedDobonCalculator**: 手札の「数字の種類数（2種類 vs 3種類以上）」を判定し、四則演算または足し算で場の数字と一致するかをチェックする関数。
- **StateManagement**: 革命（Jバック）フラグ、禁止上がり（8/2/Joker）の監視。

## 3. 進行とUIの統合
- タイトル画面にすでに追加されている「ドボン」モードと連携させてください。
- 誰かがカードを出した際に、ポンのように割り込めるUI（コンマ数秒待機し、ドボン可能ならボタンを表示）を導入してください。
- 手札がなくなった際の「全員へ5枚補充」と、山札リセット時の「ドボン倍」処理を実装してください。

まずは、ドボンのカード強さ判定と計算ロジックのテストコード、あるいはモジュール分割案から提示してください。作業を開始してください！
