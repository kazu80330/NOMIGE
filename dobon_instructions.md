# ドボン（大富豪・ヤニブ融合版）開発指示書

## ゲーム概要
「大富豪」のカードの強さと出し方をベースにしつつ、どのプレイヤーからも「ドボン（計算）」による割り込み勝利が狙える。さらに「ヤニブ」の合計点数要素も含む究極のトランプゲーム。

## 基本ルール
1. **初期手札**: 5枚。
2. **カードの強さ**: `3 < 4 < ... < 10 < J < Q < K < A < 2 < Joker`
   - **Jバック（強さ逆転）**: `J` が出ると強さが反転し、`3` が最強、`2/Joker` が最弱となる。
3. **出し方（大富豪形式）**:
   - 前の人が出したカードより「強い数字」を1枚出す。
   - 前の人が「2枚出し（ペア）」をした場合、自分も「強い数字の2枚出し」で対抗する。
   - **スート縛り**: 前と同じマークを出した場合、そのスートしか出せなくなる（Jokerは無視可能）。
   - 出せない場合は山札から1枚引く。
4. **役札（特殊カード）**:
   - `8`: 8切り。その場で流し、自分のターンで自由なカードから再開。
   - `J`: Jバック（強さ反転）。
   - `♠3`: スペ3返し。Joker単体に対してのみ割り込んで場をリセットし、自分のターンで再開。

## ドボン（割り込み）ルール
誰かがカードを出した瞬間に、手番に関係なく宣言できる（麻雀のポンのタイミング）。

1. **ドボン成功の条件**:
   - **手札の数字が2種類の場合**: 二つの数字の間で四則演算（＋−×÷）を行い、場の数字と一致すれば成功。
     - *例：場が9、手札が{3, 3, 3, 1}なら (3*3) = 9 や (3*3/1) = 9 で成功。*
   - **手札の数字が3種類以上の場合**: すべての合計（足し算）のみ行い、場の数字と一致すれば成功。
2. **カードの点数**: `A=1, 2-10=数字、J=11, Q=12, K=13, Joker=0`
3. **複数ドボン（W/トリプル）**:
   - 出したプレイヤー1名に対し、複数人が同時にドボン可能。出した人は全員に負ける。
4. **ドボン返し**:
   - 自分の捨て札に対してドボンされた際、自分の「残りの手札」で計算結果が一致すれば「ドボン返し」で全員を逆転して勝利できる。

## 勝敗と終了条件
- **あがり（勝利）**: 手札が0枚になった状態。
- **禁止あがり（負け）**: 最後の1枚が `8, 2, Joker` の場合、上がれずに「負け」となる。
- **ドロー（引き分け）**: ドボンも禁止あがりもなく、全員が手札を出し切った状態。
- **負け**: 禁止あがりをした人、またはドボン（ドボン返し）をされた人。
- **次ラウンド**: 敗者（最後に手札を持っていた人、またはドボンされた人）から開始。
- **補充**: 誰かがあがった（またはドボンが発生した）ら、全員新たに5枚引いて継続。
- **倍倍**: 山札がなくなり捨て札をシャッフルして戻すと、次のドボンの点数が2倍。

## 実装モード
1. **Yaniv**: 従来通り。
2. **Dobon**: 上記の新ルール。
3. **Yaniv-Dobon**: ヤニブの進行に、上記「ドボン（計算）」の割り込みを加えたハイブリッド。
